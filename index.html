Projects

Visualizing US International Trade

This project features a web-based data dashboard visualizing trade data by state.

It was built using d3.js and is powered by the data from the U.S. Census Bureau, specifically the Import and Export Merchandise trade statistics from 2014 to 2018.

This application provides an interactive view of the trade activity each state is engaged in, aimed at giving insight into what commodities states are trading as well as who their top trade partners are.
